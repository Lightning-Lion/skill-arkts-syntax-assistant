# 模拟交互

在模拟器中模拟用户的点击和滑动操作。

## 前提条件

需要具备良好的视觉能力，能够从截图中识别按钮的像素坐标。坐标系以屏幕左上角为原点 (0, 0)。

## 点击操作

### 基本语法

```bash
hdc shell uinput --touch --click <x> <y>
```

### 示例

点击坐标 (x=658, y=2600) 处的按钮：

```bash
hdc shell uinput --touch --click 658 2600
```

## 滑动操作

### 基本语法

```bash
hdc shell uinput -T -g <起始x> <起始y> <结束x> <结束y> <持续时间1> <持续时间2>
```

参数说明：
- `<起始x> <起始y>`: 滑动开始的坐标
- `<结束x> <结束y>`: 滑动结束的坐标
- `<持续时间1>`: 通常设置为 500（毫秒）
- `<持续时间2>`: 通常设置为 1000（毫秒）

### 示例

以下示例假设设备尺寸为 466×466 像素。

#### 从下往上滑动（向上滚动）

```bash
hdc shell uinput -T -g 233 233 233 0 500 1000
```

说明：从屏幕中心 (233, 233) 滑动到顶部中心 (233, 0)

#### 从上往下滑动（向下滚动）

```bash
hdc shell uinput -T -g 233 0 233 233 500 1000
```

说明：从顶部中心 (233, 0) 滑动到屏幕中心 (233, 233)

## 使用流程

1. **截图** - 使用截图功能获取当前界面
2. **识别坐标** - 在截图中找到目标按钮或区域的像素坐标
3. **执行操作** - 使用 uinput 命令模拟点击或滑动
4. **验证结果** - 再次截图或查看日志确认操作效果

## 注意事项

- 坐标必须在屏幕范围内，否则操作无效
- 不同设备的屏幕尺寸不同，需要根据实际设备调整坐标
- 滑动的持续时间参数（500、1000）通常不需要修改，除非需要特殊的滑动速度
